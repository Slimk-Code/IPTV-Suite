v1.0.0 — IPTV Playlist Manager & Stream Player

2026-07-03
Added
  • Foundation for IPTV playlist management and stream playback
  • Three import methods: M3U (URL or file upload), Xtream Codes API (server + credentials), Stalker Portal (MAC-based)
  • Home page with playlist dashboard (recently watched, all playlists grid)
  • Favorites (show favorite channels)
  • Playlist management (create, browse, open, delete and more)
  • Channel browser with Live/VOD/Series segmentation, category groups, and paginated channel list
  • Native video player with HLS/MPEG-TS playback (mpegts.js), SRT/VTT subtitle loading, and stream URL copy/share
